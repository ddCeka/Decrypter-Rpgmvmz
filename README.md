# Decrypter-Rpgmvmz

A command-line tool to decrypt encrypted assets in RPG Maker MV/MZ games.

## Usage

```shell
decvz <game_dir>
```

## Decryption Rules

| Encrypted File | Decrypted As |
| -------------- | ------------ |
| .rpgmvo        | .ogg         |
| .rpgmvm        | .m4a         |
| .rpgmvp        | .png         |
| .ogg\_         | .ogg         |
| .m4a\_         | .m4a         |
| .png\_         | .png         |
